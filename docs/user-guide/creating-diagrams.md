# Creating Diagrams

## Block Palette

The block palette on the left side of the window contains all available blocks organized by category.

## Adding Blocks

- **Drag and Drop**: Drag a block from the palette onto the canvas
- **Double-click**: Double-click a block in the palette to add it at the center

## Connecting Blocks

1. Click on an output port (right side of a block)
2. Drag to an input port (left side of another block)
3. Release to create the connection

## Editing Block Parameters

1. Double-click a block to open its parameter dialog
2. Modify the parameters as needed
3. Click OK to apply changes

## Selecting Blocks

- **Single selection**: Click on a block
- **Multiple selection**: Hold Ctrl/Cmd and click blocks
- **Box selection**: Click and drag on empty canvas to select multiple blocks

## Moving Blocks

- Drag selected blocks to move them
- Hold Shift for constrained movement (horizontal/vertical)

## Deleting Blocks

- Select blocks and press Delete key
- Or right-click and select "Delete"

## Subsystems

Create subsystems to organize complex diagrams:

1. Select blocks to group
2. Right-click and select "Create Subsystem"
3. Double-click the subsystem to edit its contents
